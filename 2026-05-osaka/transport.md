# 交通總整理

## 結論先講

**這趟最划算的組合是 `ICOCA + 個別購票`**，不需要買 Kansai Thru Pass。

理由：你這趟主要在「南海空港線 + 大阪市內地鐵 + 一次 JR 環狀線到 USJ」，這幾段個別買票加起來都比通票便宜，而且 ICOCA 嗶卡又快又彈性。

**住宿起點**：5/28–5/30 住難波駅周邊民宿，所有行程皆以難波為起點計算。

## 每日交通需求估算

| 日期 | 路線 | 主要工具 | 估計花費 (JPY) |
| --- | --- | --- | --- |
| 5/28 | KIX → 難波 | 南海空港急行 | 930 |
| 5/29 | 難波 ⇄ USJ（經西九条） | 地鐵 + JR | ~740 |
| 5/30 | 難波 ⇄ 千里中央 ⇄ 勝尾寺、難波 ⇄ 中崎町 | 御堂筋線 + 阪急巴士 | ~1,800 |
| 5/31 | 難波 → 臨空城 | 南海空港急行 | 890 |

| **合計** | | | **~4,400** |

> 對照 Kansai Thru Pass 2 日券 5,000 JPY、3 日券 5,800 JPY，**買票反而虧**。

## 票券方案比較

### ICOCA（推薦 ⭐⭐⭐⭐⭐）

- 嗶卡用，到處可用（南海、JR、地鐵、便利商店、自販機）
- 售價 2,000 JPY（含 1,500 可用 + 500 押金）
- 退卡可退押金（KIX 有退卡櫃台）
- **iPhone 用戶**：可直接用 Apple Wallet 加值，無實體卡

### Kansai Thru Pass（不推薦此行程）

- 2 日券 5,000 / 3 日券 5,800
- 適合「會大量搭乘私鐵、巴士跨城市」的行程（如京都、奈良、神戶、和歌山都跑）
- 你只在大阪市內 + 一次 KIX 來回，**用不到本**

### Haruka 特急（不推薦）

- JR 西日本機場特急，KIX → 天王寺 / 新大阪
- 你住難波，**搭 Haruka 還要轉地鐵**，不如直接南海

## 各段詳細路線

### 5/28 KIX → 難波

- **首選**：南海空港急行（45 分 / 930 JPY）
- **舒適選**：南海特急 Rapi:t（38 分 / 1,490 JPY，需另購特急券）
- **末班**：南海普通車約 23:55，特急 Rapi:t 22:55

### 5/29 難波 ⇄ USJ

- **去程**：難波站 → 御堂筋線 → 梅田 → JR 大阪 → 環狀線 → 西九条 → 夢咲線 → ユニバーサルシティ駅
- **更直接**：JR 難波 → 環狀線 → 西九条 → ユニバーサルシティ駅
- 車程約 30 分，票價 IC 卡 ~370 JPY 單程
- 末班車回程約 23:30

### 5/30 早上 難波 → 勝尾寺

1. 難波 → **御堂筋線（北大阪急行）** → 千里中央（約 30 分 / 380 JPY）
2. 千里中央巴士總站 5 號乘車處 → **阪急巴士 29 系統** → 勝尾寺（約 30 分 / 550 JPY）
   - **僅週六、週日、國定假日運行**（5/30 是週六 ✅）
   - 約 1 小時 1 班，**出發前一晚務必查當日時刻表**：
     - 阪急巴士官網 → 千里中央 → 勝尾寺
3. 回程同路反向

### 5/30 下午 勝尾寺 → 中崎町

- 千里中央 → 御堂筋線 → 中津 / 梅田 下車（10 分 / 320 JPY）
- 步行 5–10 分鐘到中崎町範圍
- 或搭谷町線到「中崎町」站

### 5/30 晚上 中崎町 → 道頓堀/難波

- 御堂筋線：梅田 → 難波（9 分 / 240 JPY）

### 5/31 難波 → 臨空城

- 南海空港急行：難波 → りんくうタウン（38 分 / 890 JPY）

### 6/1 Star Gate Hotel → KIX（回程 09:00 班機）

| 方案 | 時間 | 費用 | 說明 |
| --- | --- | --- | --- |
| **飯店免費接駁巴士**（首選） | ~5–10 分 | 0 | check-in 當天詢問時刻 |
| **JR 関西空港線** | 6 分 | 380 JPY | Rinku-town → KIX，班次穩定 |
| **Nankai 電鐵** | ~8 分 | ~460 JPY | 同站出發，班次亦多 |
| 計程車 | 5–10 分 | ~2,000 JPY | 備案中的備案 |

> 09:00 起飛 → 飯店 06:30 出發 → 06:45 抵達 KIX，時間充裕無壓力。
