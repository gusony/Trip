## 行李清單

### 證件 & 重要物品（隨身）

- [ ] 護照（效期 > 6 個月）
- [ ] 機票（手機 + 紙本備份）
- [ ] 訂房確認信（手機 + 紙本備份）
- [ ] USJ 票券 + 快速通關 QR
- [ ] 信用卡 × 2（建議不同卡組織，VISA + JCB）
- [ ] 日圓現金 2~3 萬
- [ ] ICOCA / 一卡通 / iPhone 加值
- [ ] 駕照國際譯本（若考慮租車，可有可無）

### 3C

- [ ] 手機 + 充電線
- [ ] 行動電源（**≤ 100Wh / 容量 < 27,000mAh** 才能上機）
- [ ] 萬國轉接頭（日本插座是 A 型雙腳，與台灣相同其實不用，但偶有圓孔）
- [ ] 相機 + 備用電池 + 記憶卡
- [ ] 耳機

### 衣物（5 月底大阪：日 22–28°C，夜 16–22°C）

- [ ] 短袖 T × 5
- [ ] 薄長袖 / 襯衫 × 2（早晚溫差）
- [ ] 薄外套 / 風衣 × 1
- [ ] 長褲 × 2、短褲 × 1
- [ ] 內衣褲 × 5+
- [ ] 襪子 × 5+
- [ ] 睡衣 1 套
- [ ] **舒適好走的鞋**（USJ 一天走 2 萬步）
- [ ] 拖鞋（住宿基本有，但個人偏好可帶）

### 衛生 / 醫療

- [ ] 個人藥品（止痛、腸胃、過敏、OK 蹦）

- [ ] 口罩
- [ ] 衛生紙、濕紙巾
