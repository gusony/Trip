# Day 2 · 5/29（五）環球影城 + 生日大餐 🎂

> 今天的關鍵：**早起、衝瑪利歐區、用快速通關吃下重點設施、晚上回難波吃生日大餐**

## 時間軸

| 時間 (JST) | 行程 |
| --- | --- |
| 06:30 | 起床、早餐（便利商店即可） |
| 07:00 | 出發前往 USJ |
| 07:40 前 | 抵達 USJ 正門，等待開園 |
| 08:00–09:00 | 開園衝瑪利歐區（如快速通關沒含 SNW，需先抽整理券） |
| 09:00–18:00 | 依快速通關時段跑設施，中間穿插用餐 |
| 18:00–20:00 | 看遊行 / 收尾紀念品 |
| 20:00–21:00 | 回難波 |
| 21:00 起 | 🎂 **生日大餐** |

## USJ → 難波 交通

- **去程**：難波 → 大阪駅（地鐵御堂筋線 9 分 / 240）→ JR 大阪環状線 → 西九条換 JR 夢咲線 → ユニバーサルシティ駅（共約 30 分，IC 卡 ~200）
- **回程**：反向，末班車約 23:30
- 或從難波直接搭 JR 難波 → 西九条 → ユニバーサルシティ（不用換地鐵）

## 入園策略

**前提**：你已有 Express Pass，但要先確認：

- [ ] 快速通關張數（4 / 7 設施版本？）
- [ ] 是否含 **超級任天堂世界（SNW）指定入場券**
  - 若**有含**：可在 Express 指定時段直接入場
  - 若**沒含**：開園後立刻去 SNW 拿「整理券」（Area Timed Entry），或衝早鳥入場（要早起＋有 Express Pass Premium / 早鳥票才行）

## 🎂 生日大餐（晚上 21:00 後）

**候選餐廳請見 [`restaurants.md`](./restaurants.md)**，現在這個時間點（出發前 8 天）建議：

- ✅ **本週內訂位**：高級燒肉、和牛料理、河豚等需訂位的店
- ✅ 給店家備註「**お誕生日**」（おたんじょうび），不少店會送小蛋糕或寫名字盤

## 散場與隔天緩衝

- USJ 通常 21:00 閉園（夏季可能 22:00），確認當日營業時間
- 吃完生日大餐回住宿不晚於 24:00
- 隔天勝尾寺需早起（建議 7:30 出發）
