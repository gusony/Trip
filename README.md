## 資料夾慣例

每個行程資料夾建議包含：

- `00-overview.md` — 總覽（日期、班機、住宿、預算）
- `01-day1-*.md` ~ `0N-dayN-*.md` — 每日逐日行程
- `transport.md` — 交通方式與票券
- `restaurants.md` — 餐廳清單與訂位狀態
- `packing.md` — 行李清單與出發前 checklist
